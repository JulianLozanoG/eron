# Directors API

## Desc
REST API that obtains a Directors list with more directed movies than a range, order asc.

## Requirements
- Java 17
- Maven

## Set Up
1. Clone the repo.
2. Go to the project folder.
3. Run `mvn clean install`.

## Run
1. Run the command `mvn spring-boot:run`.

## Use
Got to `http://localhost:8080
